# 🎬 Movie Fight

> Compare two movies side-by-side and see which one wins based on ratings, box office performance, and other statistics.

Movie Fight is a small web application that allows users to search for two different movies and compare their data visually.

---

## 🚀 Features

🔍 Search for movies using an autocomplete search box  
🎬 Compare two movies side-by-side  
📊 Display key movie statistics (ratings, box office, awards, etc.)  
🖼 Show movie posters and details  
⚡ Fetch real movie data from a public movie API  

---

## 🛠️ Built With

| Technology | Purpose |
|-----------|--------|
| 🌐 HTML | Structure of the application |
| 🎨 CSS | Styling and layout |

| 🎥 Movie API | Fetch movie data |
